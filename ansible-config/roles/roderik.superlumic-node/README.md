# superlumic-node

Ansible role to install NodeJS on OSX. This role is part of the Superlumic project that aims to simplify repeat computer setups on OSX, 10.10 and up.

## Requirements

* OSX 10.10 or 10.11

## Dependencies

* [roderik.superlumic-homebrew](https://github.com/superlumic/ansible-role-homebrew)

# Usage

Check [Superlumic](https://github.com/superlumic/superlumic) for documentation
